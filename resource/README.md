# resource 资源文件夹说明

本文件夹集中存放色色地牢的静态资源定义与引用，不包含图片二进制文件本身（图片多为外链或相对路径）。

## 文件说明

| 文件                 | 用途                                                                                                                                                                                          |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **illustration.js**  | 角色立绘。通过 `window.CHARACTER_PORTRAITS` 暴露「角色名 → 图片 URL」的映射，供界面显示角色头像/立绘。支持完整 URL 或短链（短链会自动补 `https://files.catbox.moe/` 前缀）。                  |
| **animations.js**    | 战斗动画资源。通过 `window.ANIMATIONS` 提供 APNG 动画的 key → URL 映射（斩击、恢复、受击、治疗等）；`window.ANIMATION_DURATIONS` 为各动画的播放时长（毫秒），用于战斗界面中的技能与受击表现。 |
| **svg.js**           | 界面用 SVG 图标。定义技能图标、按钮图标等内联 SVG（如攻击剑、防御盾、狼式旋风、白牙等），通过导出的变量供战斗/角色卡界面使用。                                                                |
| **svg_minified.txt** | 上述 SVG 的压缩/备份文本，便于版本管理与替换。                                                                                                                                                |

## 短链处理（立绘）

在 **illustration.js** 中，角色立绘的值可以是：

- **完整 URL**：如 `https://files.catbox.moe/jqe8hk.png`，直接使用。
- **短链**：仅文件名或路径，如 `kxlnsd.png`，不包含 `http://` 或 `https://` 时，会在加载时自动补上前缀 **`https://files.catbox.moe/`**，得到 `https://files.catbox.moe/kxlnsd.png`。
- **空字符串**：表示暂无立绘，不补前缀。
